Progress bars with tqdm

Brian Baquiran

2017-11-19 07:27

Comments

Source

At your shell:

$ pip install tqdm

Whenever you iterate over something, wrap it in a tqdm like so:

In [1]: from tqdm import tqdm

In [2]: from time import sleep

In [3]: for i in tqdm(range(1000)):
   ...:     sleep(0.01)
   ...:
